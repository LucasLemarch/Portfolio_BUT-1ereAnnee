En partant d'une besoin exprimé par un client à travers un modèle logique de données existant, et de nouveau besoin à étudier, il faut créer les nouvelles structures.
Cette SAÉ permet un premier contact avec l'implémentation d'une base de données.

Les productions :
- Jeu de données
- Modèle de données
- Rapport sur l'importance de ces données en entreprise
- Script de craétion de la base de données ( table+data)
- Script des différents états demandés
- Présentation orale

Effectifs :
Équipes de 4 étudiants
